Selector and Visual Rules

Tag Name Description Note
Selectors
selector Type selector Selects all <selector> tags.
.selector Class selector Selects all elements with class="selector".
#selector ID selector Selects all elements with id="selector".
Selector manipulation
CSS selectors can be chained so that rule sets apply only to elements that match all criteria.
type.cl Select "type" elements with the "cl" class.
.cl1.cl2 Select elements with the cl1 AND cl2 class.
CSS descendant selector
div p { } Descendant Is used to match elements that are descended from another matched selector.
!important rule
!important rule is used on declarations to override any other declarations for a property and ignore selector specificity.
Typography
@font-face {} Allows external fonts or font files to be imported directly into stylesheets.
font-family: Name of the external font.
src: Allows external fonts or font files to be imported directly into stylesheets.

Basic styling

Property Name Description Note
General
font-family: Font type Used to specify the typeface in a rule set.
font-size: Font size Used to set text sizes.
font-weight: Boldness Used to set the weight (boldness) of text normal/bold
text-align: Alignment Used to set the text alignment of inline contents. left / center / right
color: Text color Used to set the foreground text color of an element.
background-color: Background color Used to set the background color of an element.
background-image: Background image Sets the background image of an element. url("../resources/image.png");
opacity: Opacity Controls the transparency of an element.

Display and positioning

Property Values Description Note
z-index: specifies how far back or how far forward an element will appear on a web page
position: Provides options for positioning HTML elements on a page.
fixed Element stays the same regardless of scrolling. often used for nav bar.
absolute Removes an element entirely from the document flow.
relative Enables an element to be positioned relative to where it would have originally been on a web page.
display: Display Determines the type of render block for an element.
block Elements take up the full width of their container with line breaks before and after. height and width ARE adjustable
inline Elements ake up as little space as possible, flow horizontally. height and width ARE NOT adjustable
inline-block Elements can appear next to each other. height and width ARE adjustable
float: Determines how far left or how far right an element should float within its parent element.
left Content will float to the left side of the container.
right Content will float to the right side of the container.
clear: Specifies how an element should behave when it bumps into another element within the same containing element.
left No elements within the same container are allowed to float on the left side of this element.
right No elements within the same container are allowed to float on the right side of this element.
both No elements within the same container are allowed to float on either side of this element.
none Elements within the same container are allowed to float on both side of this element.

Flexbox

Property Values Description Note
display: Determines the type of render block for an element.
flex Sets an HTML element as a block level flex container which takes the full width of its parent container
inline-flex Sets an HTML element as an inline flex container which takes only the required space for the content.
flex-direction: Specifies how flex items are placed in the flex container. Vertically or horizontally.
flex-wrap: Specifies whether flex items should wrap or not. Becomes a priority over shrinking. Applies to flex items only.
Justify content
justify-content: Defines how the space is distributed between and around content items along the main-axis of their container. (horizontally)
flex-start
flex-end
center
space-between
space-around
Flex properties
flex-grow: Allows flex items to grow as the parent container increases in size horizontally.
flex-shrink: Allows flex items to shrink as the parent container decreases in size horizontally.
flex-basis: Sets the initial base size for a flex item before any other space is distributed according to other flex properties.
flex: Grow, shrink and basis declared on one line.
Flex-wraps
flex-flow: Provides a shorthand for the properties flex-direction and flex-wrap. Applies to flex items only.
align-content: Modifies the behavior of the flex-wrap property. It determines how to space rows along the cross axis (vertically)
align-items: Used to align flex items vertically within a parent container.
