Semantic Placement Tags

S-C Tag Name Description Note
X <header> Header The wrapper element for the header of your page. (max 1).
X <main> Main The wrapper element for the body of your page. (max 1).
X <footer> Footer The wrapper element for the footer of your page. (max 1).
X <nav> Nav The wrapper element for the navigation on your page (max 1 but more can be added with different aria-label).
X <section> Section The wrapper element for a section of your page.
X <article> Article The wrapper element for an article inside a section.
X <aside> Aside The wrapper element for an aside note inside a section.

Embedding Media Tags

S-C Tag Name Description Note
X <video src=""> Video Allows to add video.
X <audio src=""> Audio Allows to add video.
V <embed src=""/> Embed Allows to add any type of media. Self-closing, ATTENTION for the ending / here.
X <figure> Figure Allows to encapsulate images, diagrams or code snippet.
V <img src=""> Image Allows to add image URL. Self-closing.
X <figcaption> Figcaption Allows to add description that will follow along with the media.

Element Tags

S-C Tag Name Description Note
Lists
X <ol> Ordered list Creates a list with numbers.
X <ul> Unordered list Creates a list with dots.
X <li> List item Creates an entry in a list.
Basic styling
X <h1> - <h6> Heading element The heading elements are ordered from the highest level, 1, to the lowest level, 6.
X <span> Span element An inline container for text, makes precise styling with a class/id.
X <p> Paragraph Contains and displays a block of text.
V <br> Line Break Creates a line break in text and requires only an opening tag. Self-closing.
X <em> Emphasis Emphasizes text and browsers will usually italicize the emphasized text by default.
X <strong> Strong Highlights important or urgent text and browsers will normally render this highlighted text in bold.
Link making
X <a href="link"> Anchor Creates hyperlinks on what's linked inbetween opening and closing tag. Can be internal with #id instead of link
X <a target=""> Target attribute Specifies where a hyperlink should be opened. _blank being new tab.

Tables tags

S-C Tag Name Description Note
X <table> Table Element Used to represent a two-dimensional table made of rows and columns. Can't have both <tbody> and <tr> as direct children.
X <tr> Table Row Used to add rows to a table before adding table data or inside table headings
X <th> Table Heading Used to add titles to rows and columns of a table
X <td> Table Data Used to add a cell of data to a table.
Structure of a table
X <thead> Table Head element Defines the headings of table columns encapsulated in table rows.
X <tbody> Table Body element Is a semantic element that will contain all table data other than table heading and table footer content.
X <tfoot> Table Foot element Uses table rows to give footer content or to summarize content at the end of a table.
Expansion of cells
X <colspan> Column Span Defines how many columns a td element should span.
X <rowspan> Row Span Defines how many rows a td element should span.
