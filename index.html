HTML5 Reference

Semantic Placement Tags

S-C Tag Name Description Replaces former
X <header> Header The wrapper element for the header of your page. (max 1). <div> for more precision.
X <main> Main The wrapper element for the body of your page. (max 1). <div> for more precision.
X <footer> Footer The wrapper element for the footer of your page. (max 1). <div> for more precision.
X <nav> Nav The wrapper element for the navigation on your page (max 1 but more can be added with different aria-label). <div> for more precision.
X <section> Section The wrapper element for a section of your page.
X <article> Article The wrapper element for an article inside a section.
X <aside> Aside The wrapper element for an aside note inside a section.

Embedding Media Tags

S-C Tag Name Description Replaces former
X <video src=""> Video Allows to add video.
X <audio src=""> Audio Allows to add video.
V <embed src=""/> Embed Allows to add any type of media.
X <figure> Figure Allows to encapsulate images, diagrams or code snippet.
V <img src=""> Image Allows to add image URL.
X <figcaption> Figcaption Allows to add description that will follow along with the media.
